# 前言

欢迎来到“小程序插画共享平台ssm”项目。该项目旨在为用户提供一个便捷、高效的插画共享平台，通过微信小程序实现与用户的互动。以下为项目的详细介绍。

# 内容介绍

“小程序插画共享平台ssm”是一个基于Java语言和Spring、Springmvc、Mybatis框架开发的插画共享平台。用户可以在平台上发布自己的插画作品，与其他用户互动、交流。平台提供丰富的功能，包括作品展示、分类浏览、搜索、点赞、评论等。通过微信小程序，用户可以随时随地进行操作，方便快捷。

# 技术介绍

- 语言：Java
- 使用框架：Spring、Springmvc、Mybatis，微信小程序
- 前端技术：JS、Vue、css3，Uniapp
- 开发工具：IDEA/Eclipse，Uniapp
- 数据库：MySQL 5.7/8.0
- 数据库管理工具：phpstudy/Navicat
- JDK版本：jdk1.8
- Maven: apache-maven 3.8.1-bin
- 前端环境：Node.Js 12\14\16
